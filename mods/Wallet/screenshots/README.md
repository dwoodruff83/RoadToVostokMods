# Screenshots

Source images for the ModWorkshop upload. Not packaged into the `.vmz`.

## Naming convention

| File | Purpose |
|---|---|
| `01-overview.png` | Hero / featured image. The most representative single shot. |
| `02-mcm.png` | The mod's MCM page (Configuration Menu integration). |
| `03-…` onward | Specific features, in priority order. |

PNG preferred. Aim for 1280×720 or higher; ModWorkshop downscales automatically.
